Key Terms

Responsive

The ability to render content with display based on the container it is rendered in.

Media Queries

CSS Definition containers that are dependent on a given media that is rendering the content.

Breakpoints

The definitions, based on width, where media queries kick in to drive style changes.

Flow

The manner/ruleset in which the browser renders the document.

What they are not.

Frameworks are not typically going to be used in a web product for

  • The final overall look or theme of a production site.
  • Covering the full gamut of website interaction.
  • Going to make or break any web property.
  • Magic.
They are the same base of CSS you'd use for a custom project, but pre-baked to make life easier. You still have to know what you are doing.

Problem Solvers

Ulitmately, frameworks exist to solve a set of problems.

  • Quickly Prototyping Solutions
  • Cross-Screen Consistency
  • Enforcing Design Consistency
  • Creating Maintainable Templates ("Futureproofing")
  • Hide Crazy CSS Workarounds
Mention that a good codebase looks like it was written by one developer. Ultimately, they make collectinos of boxes that have variable layouts based on their context.

Opinionated

There are nuances in how they solve these problems.

  • Grid Dimensions, Fluid/Stiff Grids
  • Normalization: font-size, line-height, column-width, gutters
  • Integrated Javascript
  • Number of Breakpoint sets (small, med, large)
  • Icons/Fonts
  • UI and (Forced) Customization
  • Which Browsers to Support
